ARIA 属性类似于 ARIA 状态，但是在页面上是相对静态的，它们作为 HTML 元素的其他属性。 小组件属性与小组件状态相似，但是该值在页面范围内保持不变。 有 11 个全局属性和 14 个小组件属性（请参阅图 3）。

语义 HTML 表示页面中的 HTML 标记应以与其含义而非表示有关的方式描述内容。 有关内容修饰的所有信息应放入 CSS 文件， 而应在加载页面和分析并应用样式表后通过外部链接的 JavaScript 添加网页的逻辑和客户端行为。 逐渐增强确保了 JavaScript 文件出错时，仍可以正确样式加载页面。 如果忽略 CSS 文件（例如，通过屏幕阅读器），HTML 页面仍包含所有内容。

残障人士辅助功能旨在提供以下四个主要方面的解决方案：

  • 听力
  • 移动性
  • 认知能力
  • 视觉

听力问题是指用户可能无法听到网站上的任何声音。 其解决方法是针对所有非文本内容（如字幕和隐藏字幕）使用替代文本来帮助用户感知内容。 这包括改写成文字的言语和符号语言（如果可能）。
